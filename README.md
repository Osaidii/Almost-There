# Almost There. 
A Horror Mystery Game in Production.  
The game revolvs around a person who arrives for a job interview in a lonely and eerie place. Suddenly, he faints and wakes up in a wierd yellow place. He must find his way out.

### The Game is in Early Access.   

## Controls
1. WASD to Move. 
2. Mouse To Look Around.
